# David Fincher Tribute

A high-end interactive web experience focused on advanced motion design and custom navigation patterns.

### 🛠 Engineering Highlights
* **Custom Carousel Engine:** Implemented a math-based horizontal slider from scratch, handling mouse/wheel delta events for a smooth "inertia" feel.
* **Dynamic Viewport Calculations:** Real-time calculation of `rectHeight` and `rectTravel` for a "minimap" scroll indicator (About Section).
* **Motion Design:** Fine-tuned animations using `framer-motion` with custom cubic-bezier easing for a cinematic feel.
* **Performance:** Optimized layout shifts and used `AnimatePresence` for seamless state transitions.

### 🏗 Tech Stack
Next.js 13, TypeScript, Framer Motion, Tailwind CSS.
